# CaMut: Causal Analysis of Mutations

CaMut is a computational framework that implements a hybrid causal inference approach, integrating constraint-based and score-based algorithms, to construct gene networks from somatic mutation data. This specific version was developed to identify gene networks in Diffuse Large B-Cell Lymphoma (DLBCL) datasets.

---

## How to Run

1. Open CaMut.ipynb in **Google Colab**.
2. Run the first cell and upload your DLBCL mutation data in MAF format.
3. Run the remaining cells sequentially.
4. View the files with graphs and other information provided by CaMut in the Output folder.
